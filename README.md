## 1. Introduce 🎉

- ### What's 42polar?
 　 　``` 42서울에서 진행되는 멘토링 서비스를 이용할 수 있도록 도와주는 웹사이트입니다.```

- ### User

> **42cadet** : 멘토 검색 및 멘토링 신청 <br> 
> **42mentor** : 멘토링 수락 및 보고서 작성 <br> 
> **42bocal** : 보고서 확인 및 출력 <br>
